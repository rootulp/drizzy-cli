# drizzy-cli

> Prints Drake lyrics using [drizzy-server](https://github.com/rootulp/drizzy-server)

## Install

```
$ npm install --global drizzy
```

## Usage

```
$ drizzy --help

  Prints Drake lyrics using drizzy-server

  Usage
    $ drizzy
```
